# docs/launch — launch & distribution material

Public-facing launch content for JVoice (a free, open-source macOS dictation app).

**Do not publish anything here without David's explicit go-ahead.** No posting to Hacker News,
Reddit, or directories; no `gh repo create`, `git push`, or adding git remotes. David posts these
personally (Hacker News bans AI-written replies).

## Files
- `distribution-plan.md` — the researched channel playbook (r/macapps → Show HN → awesome-lists →
  directories; TikTok for non-technical users).
- `unsigned-distribution-findings.md` — the realities of distributing an unsigned app: the
  Gatekeeper "Open Anyway" flow, the DMG-not-zip rule, the Homebrew unsigned-cask ban, and the
  VoiceInk free-source/paid-binary model.
- `launch-post-drafts.md` — Hacker News / Reddit / press drafts.

## Conventions
- `USER` is a literal placeholder for the GitHub username in `README.md` and the drafts — replace it
  at publish time.
- $0 budget: distribution is an unsigned, ad-hoc-signed DMG via GitHub Releases (no Apple Developer
  account); hosting is free (GitHub Pages). Keep everything free.
