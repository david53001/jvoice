## What I applied (Tier 1 — real defect + locally verifiable)
1. **Custom-word corruption fixed end-to-end** (`fix(text)` + `fix(phonetic)`). `.NET` was becoming
   `..NET`/`...NET` — a double/triple substitution across `applyCorrections` AND `PhoneticMatcher`. Both
   closed; `process("use dot net daily", vocab:[".NET"])` now yields `use dot .NET daily`.
2. **`[A]`/`[I]`/`[II]`/`[X]` no longer deleted** — `stripDecoderArtifacts` now only strips real decoder
   sentinels (underscore-bearing or an allow-list keyword).
3. **Paste privacy** — the transcript pasteboard item is now marked Concealed/Transient/AutoGenerated so
   clipboard-history managers skip it (Cmd+V still works). Partial mitigation only (Universal Clipboard has
   no public opt-out — full fix is Tier 2).
4. **Last transcript is clearable** — `Clear` button in Settings + cleared on Reset (was plaintext in prefs
   forever, surviving reset). Corrupt-settings backup also cleared on explicit reset.
5. **Lifecycle/leaks** — frontmost observer removed in `deinit`; in-flight WAV cleaned up on *any*
   termination (not just menu Quit); redundant hotkey re-registration removed.
6. **Validated custom-word input** — length cap, must contain a letter/digit, case-insensitive dedupe
   (garbage entries used to pollute the decoder prompt). Comma-splitting deliberately deferred (your call).
7. **CI actually guards the app now** — runs `swift build -c release`, runs both logic scripts, and the
   test-count gate tracks the authored count (≥ authored−5) instead of a stale `90`. WhisperKit pinned
   `exact: 1.0.0`. Script diagnostics unhidden.
8. **New tests** — StatsStore, target-PID resolution (3 branches), AX-prompt predicate, revert round-trip,
   paste success/coalescing, menu-bar icon states, + regression cases for every fix above.
