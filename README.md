# JVoice

JVoice is a menu-bar-only macOS 14+ voice dictation app: press ⌥Space to record, transcribe on-device with [WhisperKit](https://github.com/argmaxinc/WhisperKit), tone-style the result, and paste it into the frontmost app. Built as a SwiftPM executable; this README is a placeholder and will be expanded later.
